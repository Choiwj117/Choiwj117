# Day 1

### 1. Project Overview
- 데이터셋 이름: Simulated Roulette Data
- 주요 목표: 10000회 동안의 룰렛 게임중 임의의 베팅시의 승리 확률 확인
- 데이터 출처: Kaggle

### 2. Day 1 Goals
- 데이터셋 구조 파악 (변수 확인, 데이터 타입, 결측값 등)
- 주요 변수 탐색 (EDA - Exploratory Data Analysis)

### 3. Tasks for Today
1. 데이터 로드 및 기본 정보 확인
    - `head()`, `info()`, `describe()` 사용
2. 결측값 확인 및 처리 방안 계획
    - `isnull().sum()`
3. 변수별 분포와 상관관계 분석
    - 히스토그램, 상관 행렬 등
4. 간단한 시각화 생성
    - `matplotlib`와 `seaborn` 활용
